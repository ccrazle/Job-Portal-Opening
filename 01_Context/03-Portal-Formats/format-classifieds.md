# Classified Ads — Job Posting Format & Best Practices

## Overview

Classified ads cover local/regional job boards (Quikr Jobs, OLX Jobs) and newspaper classifieds. These postings are compact, print-optimized, and designed for maximum information density in minimal space.

## Required Fields

| Field | Description | Notes |
|-------|-------------|-------|
| Job Title | Standard role title | Short, keyword-rich |
| Company Name | Organization name | |
| Location | City and area/locality | Specific area is critical |
| Salary | Monthly salary range | Always include — it's the primary hook |
| Experience | Minimum years required | Keep it simple |
| Contact | Phone number and/or email | Primary response channel |
| Job Description | Brief role summary | 500–800 characters, print-optimized |

## Best Practices

1. **Front-load critical info** — Title, Salary, Location in the first line. Readers scan quickly.
2. **Keep it under 800 characters** — classified space is expensive and attention is short
3. **Include phone number prominently** — many candidates in this segment prefer calling over email
4. **Use abbreviations common in classifieds** — "Exp: 2-5 yrs", "Sal: 15-20K/month", "Loc: Mumbai"
5. **One role per ad** — don't combine multiple positions in a single classified
6. **Mention "immediate joining" if applicable** — urgency drives response rates
7. **State interview location and process** — "Walk-in interview at {address}" performs well
8. **Avoid long qualification lists** — keep it to 2-3 key requirements
9. **Regional language versions** — for local newspaper classifieds, consider Hindi/regional language adaptations
10. **Include timing** — "Contact between 10 AM - 6 PM" reduces off-hours calls

## Description Template — Online (Quikr / OLX Jobs)

```
{Job Title} | {City, Area} | {Monthly Salary Range}

{Company Name} is hiring {Job Title} for {Location}.

Requirements:
- Experience: {Range}
- Skills: {Key skill 1}, {Key skill 2}
- Education: {Minimum qualification}

Responsibilities:
- {Primary duty 1}
- {Primary duty 2}
- {Primary duty 3}

Salary: {Monthly Range} + {Benefits if any}
Timings: {Work hours}

Contact: {HR Name} — {Phone Number}
Email: {HR Email}
Walk-in: {Address, if applicable} | {Date/Time, if applicable}
```

## Description Template — Print (Newspaper)

```
WANTED: {JOB TITLE}
{Company Name}, {City}
Exp: {Range} | Sal: {Monthly Range}
{1-line role summary}
Req: {Key qualifications, comma-separated}
Contact: {Phone} / {Email}
{Walk-in details if applicable}
```

## Character Limits

| Format | Limit |
|--------|-------|
| Online (Quikr/OLX) | 800 characters |
| Print (Newspaper) | 500 characters (approx. 8-10 lines) |

## Target Roles

Best suited for: Site Supervisors, Electricians, Mechanics, Technicians, Helpers, Store/Warehouse staff, Security, Drivers, and other field/trade roles where candidates rely on local job boards and newspaper classifieds.
