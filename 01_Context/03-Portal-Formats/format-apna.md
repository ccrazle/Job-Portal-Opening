# Apna Jobs — Job Posting Format & Best Practices

## Required Fields

| Field | Description | Notes |
|-------|-------------|-------|
| Job Title | Standard role title | Simple, direct — avoid corporate jargon |
| Company Name | Organization name | |
| Location | City | Specific area preferred for blue-collar roles |
| Salary | Monthly salary range | Candidates expect monthly figures |
| Experience | Years of experience or "Freshers welcome" | Be specific about minimum |
| Job Description | Role details | Max 1,500 characters. Simple language. |
| Skills Required | Practical/trade skills | Tag format |
| Job Type | Full-time / Part-time / Contract | |

## Best Practices

1. **Use simple, everyday language** — Apna's audience is largely blue-collar and entry-level; avoid corporate jargon
2. **Lead with salary** — for trade and entry-level roles, pay is the primary decision factor
3. **Mention location and commute** — proximity matters heavily for field/site roles
4. **List day-to-day tasks** — candidates want to know exactly what they'll be doing, not abstract responsibilities
5. **Keep it under 1,500 characters** — shorter posts perform better on mobile-first platforms
6. **Specify practical skills** — "welding", "electrical wiring", "plumbing" not "technical proficiency"
7. **Include work timings and shifts** — critical for site/field roles
8. **Mention benefits beyond salary** — accommodation, travel allowance, meals, PF/ESI
9. **Use Hindi-friendly terminology where appropriate** — many candidates are more comfortable in Hindi
10. **State physical requirements honestly** — site work, lifting, standing hours

## Description Template

```
{Job Title} — {Company Name}

Location: {City, Area}
Salary: {Monthly Range}
Experience: {Range or "Freshers welcome"}

What you'll do:
- {Day-to-day responsibility 1}
- {Day-to-day responsibility 2}
- {Day-to-day responsibility 3}
- {Day-to-day responsibility 4}

What you need:
- {Practical skill/qualification 1}
- {Practical skill/qualification 2}
- {Minimum education if required}
- {Any certifications needed}

Timings: {Shift details}
Benefits: {PF, ESI, accommodation, travel, meals — whatever applies}

To apply: {Instructions}
```

## Character Limits

| Field | Limit |
|-------|-------|
| Job Description | 1,500 characters max |
| Job Title | 60 characters |

## Target Roles

Best suited for: Electricians, Mechanics, Site Workers, Lab Technicians, Store Helpers, Data Entry Operators, Delivery Staff, Security Guards, Skilled Trades.
